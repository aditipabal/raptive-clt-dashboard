# Central Limit Theorem Dashboard

This Streamlit application demonstrates the Central Limit Theorem using interactive simulations.

## Features

- Four probability distributions
- Adjustable sample size
- Adjustable number of simulations
- Original distribution
- Sampling distribution
- Summary statistics
- Interactive controls

## Technologies

- Python
- Streamlit
- NumPy
- Pandas
- Matplotlib
- SciPy
